# Markdown 语法大全 📝

一份**中文 Markdown 语法速查手册**：每个语法都是「源码写法」+「渲染效果」上下对照，覆盖标准语法与常用扩展语法，共 17 大类。

![示例图片](images/demo.svg)

## ✨ 特点

- **一目了然**：源码与效果紧挨着排列，边看边学
- **覆盖全面**：从标题、列表到数学公式、Mermaid 图表
- **区分标准/扩展**：明确标注哪些语法需编辑器支持
- **附速查表**：常用语法一表搞定

## 🚀 使用方法

直接打开 [`Markdown语法大全.md`](Markdown语法大全.md) 查看。

> 💡 建议用 **Typora** 或 **Obsidian**（或装了预览插件的 VS Code）打开，扩展语法（高亮、上下标、脚注、公式、Mermaid）才能正常渲染。
